# Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my education, skills, projects, and experience.

## Overview

This portfolio presents my academic background, technical skills, and projects in a clean and interactive layout.
The website uses a custom color palette and smooth animations to create a professional developer profile.

## Features

* Responsive navigation bar
* Smooth scrolling between sections
* Interactive project cards with hover effects
* Fade-in animation on scroll
* Modern color palette
* Clean and readable layout
* Download resume button
* Contact information section

## Sections Included

* **Home** – Introduction and profile heading
